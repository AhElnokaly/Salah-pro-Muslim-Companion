# 🕌 تطبيق رفيق المسلم (Muslim Companion)

تطبيق رفيق المسلم هو تطبيق ويب متكامل ومصقول يهدف إلى مرافقة المسلم في عباداته اليومية من صلوات وأذكار وقراءة للقرآن الكريم، بأسلوب تصميم حديث، مريح للعين، وإيجابي يبتعد تماماً عن لغة التأنيب والتقصير ويشجع على التقدم الدائم (No-Guilt UX).

---

## ✨ الميزات الرئيسية المطورة حديثاً

### ١. 🌟 وضع يوم الجمعة الخاص (Special Friday Mode)
* يُعرض تلقائياً يوم الجمعة بلمسات جمالية خضراء وذهبية مبهجة.
* يحتوي على **قائمة السنن والآداب التفاعلية** (الغسل، التطيب، التبكير، الكهف، كثرة الصلاة على النبي ﷺ).
* **عداد مخصص للصلاة على النبي ﷺ** لتشجيع المستخدم بطريقة تفاعلية وبسيطة.

### ٢. 🕌 شاشة الأذان الكاملة والغامرة (Immersive Athan Overlay)
* شاشة تفاعلية كاملة تظهر تلقائياً عند دخول وقت الأذان أو يمكن اختبارها يدوياً من زر الجرس المخصص في الكارد الرئيسي.
* تدعم نغمة صوتية هادئة تم تخليقها بالكامل برمجياً عبر (Web Audio API) لتناسب الأجواء الروحانية.
* تعرض ترجمة لمعاني جمل الأذان باللغة الإنجليزية وتتحول تلقائياً بعد انتهاء النداء إلى **دعاء ما بعد الأذان الشريف** لتعظيم الأجر.
* إمكانية الاختيار بين أشهر المؤذنين (مكة المكرمة، المدينة المنورة، المسجد الأقصى).

### ٣. 📱 محاكي ويدجت الهاتف (Mobile Widget Simulator)
* محاكاة ذكية وصور حية للـ Widgets الخاصة بتطبيق رفيق المسلم على شاشات الهواتف الذكية.
* يدعم أحجاماً مختلفة:
  * **الصغير (٢×٢):** يعرض الورد اليومي مع تنازلي لأقرب صلاة.
  * **المتوسط (٤×٢):** شريط المواقيت الخمسة مع مؤشر تفاعلي ملون للصلاة القادمة.
  * **الكبير (٤×٤):** لوحة معلومات متكاملة تضم أذكار اليوم، ونسبة إنجاز الورد القرآني، وإحصائيات السنن اليومية.
* إرشادات عربية مفصلة لطريقة إضافة وتثبيت الـ Widgets على هواتف Android و iOS.

### ٤. 📊 شريط تقدم مواقيت الصلاة (Prayer Progress Bar)
* خط تقدم انسيابي متدرج الألوان يوضح بدقة متناهية نسبة الوقت المنقضي من الصلاة الحالية والوقت المتبقي لإقامة الصلاة القادمة بشكل هندسي دقيق يسهل فهمه في لمحة سريعة.

### ٥. ⏰ إعادة تصميم الكارد الرئيسي والساعة الرقمية
* إعادة تصميم شاملة لواجهة الكارد الرئيسي لتعمل كتحفة فنية غامرة.
* تظهر الساعة الرقمية بالأرقام العربية الأنيقة وبخط عريض يتوهج بنعومة.
* تبين بوضوح تام تفاصيل الصلاة القادمة والوقت المتبقي لها كجزء أساسي من الواجهة.

### ٦. 📖 منظومة الختمة والقرآن المرنة (Flexible Quran Tracker)
* خوارزمية ذكية تتفهم ظروف المستخدم؛ في حال فوت قراءة ورد اليوم من الصفحات، يعاد توزيع الصفحات المتبقية تلقائياً ومرونة على الأيام الباقية لتجنب تراكم الورد أو شعور المستخدم بالعجز والتقصير.

### ٧. 🤝 التدوين الإيماني الرحيم والآمن (Compassionate & Sandbox-Safe Logging)
* **واجهة تدوين إيجابية:** تغيير مسمى خيار الصلاة الفائتة في مفكرة الصلاة والسنن من "فاتتني" إلى "صليتها متأخر"، تماشياً مع فلسفة تشجيع المستخدم على استكمال الصلاة وتعزيز الحافز والهمة بعيداً عن مشاعر التقصير والتأنيب.
* **محاكي تخزين آمن (Sandbox-Safe Memory Fallback):** تم بناء طبقة حماية برمجية كاملة (Storage Polyfill) تضمن استقرار تدوين الصلوات والعبادات وحفظها بنجاح حتى في بيئات العرض المقيدة وداخل الـ iframes التابعة لمنصات البرمجة، مع توفير بديل ذكي في الذاكرة لتفادي مشكلات صلاحيات الوصول (DOMExceptions).

### ٨. 🔄 منظومة التوجيه التفاعلي والاهتزاز اللطيف (Interactive Gentle Nudge)
* **تأثيرات لفت الانتباه الدقيقة:** عند دخول وقت الصلاة وعدم تسجيلها بعد، تهتز خانة الصلاة الحالية بلطف متقطع (Gentle Wiggles) وتتوهج ببريق ذهبي دافئ لجذب نظر المستخدم وتوجيهه للضغط عليها واستكشاف التطبيق وتدوين عباداته.
* **توجيه إرشادي ذكي:** تتبدل العبارة الإرشادية أسفل كارد الصلاة بشكل حيوي لتنبيه المستخدم بالصلاة الحالية وحثه على التسجيل اليومي بلمسة بصرية ودية.

### ٩. 📲 التثبيت الفوري وتطبيق الويب التقدمي (PWA Standalone & Install Guide)
* **تثبيت التطبيق كتطبيق مستقل (PWA):** تمت إضافة زر تثبيت مخصص بالتطبيق ليتمكن المستخدم من تنزيله وتثبيته كتطبيق مستقل على هاتفه (Android / iOS) أو الحاسوب (PC / Mac) مع الحفظ المستمر وأداء سريع كالتطبيق الأصلي.
* **دليل التثبيت المرئي المتوافق:** توفير واجهة إرشادية مدمجة شديدة الأناقة تشرح خطوة بخطوة لكافة منصات التشغيل (مثل iOS Safari و Android Chrome) كيفية تثبيت التطبيق وإضافته للشاشة الرئيسية للاستخدام الفوري.

## 🎨 الهوية البصرية وفلسفة التصميم

* **النمط الإيماني الزاهي (Faith Bright Style):** ألوان مريحة ومبهجة مستوحاة من ظلال جدران المساجد البيضاء وخضرة القباب النبوية.
* **النمط الزجاجي الغامر (Glassmorphic Dark Style):** طبقات زجاجية شفافة فوق خلفيات ليلية عميقة تناسب القراءة الهادئة في الليل وتوفر طاقة شاشة الهاتف.
* **الخلفيات التلقائية الذكية:** خلفيات مساجد مدمجة تتحول تلقائياً حسب فترات اليوم (فجر، شروق، ظهر، عصر، مغرب، ليلة الجمعة، رمضان، والعيدين).

## 🚀 التجهيز للنشر على Netlify (Netlify Deployment Guide)

التطبيق مبني بالكامل كـ Single Page Application (SPA) باستخدام **Vite & React & Tailwind CSS**، مما يجعله جاهزاً ومثاليًا للنشر الفوري والمجاني على منصة **Netlify**.

### 🛠️ خطوات النشر الفوري:

#### الطريقة الأولى: عبر ربط مستودع GitHub (موصى بها للتحديث التلقائي)
1. قم برفع كود المشروع على مستودع خاص بك على GitHub.
2. توجه إلى موقع [Netlify](https://www.netlify.com/) وسجل الدخول.
3. اضغط على **"Add new site"** ثم اختر **"Import an existing project"**.
4. اختر **GitHub** وقم بإعطاء الصلاحية للوصول للمستودع.
5. سيقوم Netlify بالتعرف على إعدادات المشروع تلقائياً، تأكد من مطابقة الإعدادات التالية:
   * **Build command:** `npm run build`
   * **Publish directory:** `dist`
6. اضغط على **"Deploy Site"** وسيصبح تطبيقك جاهزاً في غضون ثوانٍ!

#### الطريقة الثانية: النشر اليدوي السريع (Drag and Drop)
1. قم ببناء ملفات الإنتاج النهائية محلياً عن طريق تشغيل الأمر التالي في جهازك:
   ```bash
   npm run build
   ```
2. سيقوم الأمر بإنشاء مجلد باسم `dist` يحتوي على ملفات HTML و JS و CSS والصور المعالجة بالكامل.
3. توجه إلى صفحة [Netlify Drop](https://app.netlify.com/drop).
4. اسحب مجلد `dist` المولد وأفلته في المربع المخصص في المتصفح.
5. سيتم رفع الموقع فوراً وسيعطيك Netlify رابطاً مباشراً لمشاركته مع أصدقائك!

## 💻 بيئة التطوير والتشغيل المحلي

إذا أردت تشغيل المشروع وتطويره على جهازك المحلي:

1. قم بتثبيت الحزم البرمجية المطلوبة:
   ```bash
   npm install
   ```
2. قم بتشغيل خادم التطوير المحلي:
   ```bash
   npm run dev
   ```
3. افتح المتصفح وتوجه إلى العنوان: `http://localhost:3000`
